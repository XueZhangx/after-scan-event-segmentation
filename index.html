이벤트 분할 연구 (Event Segmentation Study)

본 실험에 참여하기 전, 아래 내용을 주의 깊게 읽어주세요:

  • 과제: 동영상을 시청하며 사건의 경계를 구분합니다.
  • 역할: 의미 있는 사건 단위가 끝나거나 새로운 사건이 시작될 때마다 스페이스바를 누르고, 해당 사건에 대한 간단한 설명을 작성합니다.
  • 구성: 연습 영상(1개) + 본 실험 영상(16개, 2개 세션으로 나뉨, 각 세션당 8개 영상)

조작 방법 (How to Mark Events):

  • 스페이스바 (SPACEBAR) - 의미 있는 사건 단위가 끝났거나 새로운 사건이 시작된다고 생각될 때 누릅니다 (다시 누르면 계속 재생됩니다)
  • ← → 방향키 - 시간 미세 조정 (±1초)
  • 엔터 (ENTER) - 시간 확정; 설명 입력 후 엔터를 눌러 제출
  • ESC - 표시 취소 또는 입력 취소

아래 버튼을 클릭하면 위 내용을 이해했으며 실험 참여에 동의하는 것으로 간주합니다.
